Guide

To generate a random piece, click the middle button (the one with the piece names).

To view the attributions, click the top button (Con Fuoco). This will redirect you to a new tab with all the pieces and thier attributions.

If a piece's attribution is not listed, then it is public domain.

If a piece has a number with a set of parentheses, this indicates that it has an attribution. The number is the attribution #.

The format of pieces: Name - Movement (if applicable) - Composer - Performer (attribute number).

To return to the home page, close this window.

Press the space bar to hide mouse.
